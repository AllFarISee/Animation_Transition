## Implicit Animation
#### Animasi saklar lampu ON/OFF menggunakan :
  - AnimatedContainer
  - AnimatedAlign
  - perubahan warna, ukuran, dan posisi

#### Ketika user menekan saklar :
  - ukuran & warna lampu berubah
  - posisi saklar bergeser
  - efek cahaya muncul
